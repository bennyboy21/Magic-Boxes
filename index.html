How To Play
1.
Take a 3x3 box like the one on the right and fill it with numbers 1 to 9, using each number only once.
2.
The Magic Square is complete when all rows, all columns, and both diagonals add up to the same number.
3.
If you want to input a negative integer then just click/type the negative symbol and it will turn green and will make whatever number you input negative
